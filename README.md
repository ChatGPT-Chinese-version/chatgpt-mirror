### 什么是镜像网站？

镜像网站是通过复制原始网站内容并在另一个服务器上进行托管，以提供备用访问途径的一种网站。通常，镜像网站可以减轻主站点的流量压力，并确保用户的访问不会受到限制。更新频率会因镜像网站设置的不同而有所差异。
